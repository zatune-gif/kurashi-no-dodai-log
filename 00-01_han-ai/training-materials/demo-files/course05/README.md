# コース⑤ 研修用デモファイルセット

すべて架空データ（2026-07-02 生成・動作検証済み）。実在の企業・取引とは関係ありません。

## 使い方（研修当日の準備）

このフォルダ一式を講師PCのデスクトップにコピーしておく。受講者に体験させる場合は受講者PCにもコピーする（USBまたはダウンロードリンク）。

| フォルダ / ファイル | 用途 | 研修での指示文 |
|---|---|---|
| `Downloads_mock/` | ユースケース① フォルダ整理 | 「Downloads_mock にある画像・PDF・Wordを種類ごとにフォルダ分けして」 |
| `invoices/` | ユースケース② PDF情報抽出 | 「invoicesフォルダの請求書PDFから会社名・金額・日付を読み取ってCSVにまとめて」 |
| `companies.csv` | ユースケース③ 定型文書の一括生成 | 「companies.csv のリストの各社に向けた個別のお礼メールをテキストで作って」 |

## 内容

- `Downloads_mock/` — PDF2・Word2・PNG2・JPG1・MP4 1・TXT2 の計10ファイル（種類混在）
- `invoices/` — 架空の請求書PDF 5枚（社名・金額・日付・品目が1枚ずつ異なる。複数品目の請求書も含む）
- `companies.csv` — 5社の会社名・担当者・業種（UTF-8 BOM・Excelでそのまま開ける）
