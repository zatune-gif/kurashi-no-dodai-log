# コース② 研修用サンプルデータ

すべて架空データ（2026-07-02 生成）。台本 `script_course02.md` の実施前チェックに対応。

## 残っている手作業（ブラウザ操作・約10分）

1. Googleドライブで新規スプレッドシートを3つ作成し、以下のCSVの中身をそれぞれ貼り付ける
   - `survey_responses.csv` → シート名「アンケート回答」（Section 3 実践②用）
   - `product_list.csv` → シート名「商品リスト」（Section 4 実践①用）
   - `customer_list.csv` → シート名「顧客リスト」（Section 4 実践②用）
2. 各シートを「リンクを知っている全員が閲覧可」で共有し、URLを取得する
3. 取得した3つのURLをスライドの該当ページに掲載する（Section 3 実践②・Section 4 実践①②）

※CSVはUTF-8 BOM付きなので、Excelで開いてコピー→貼り付けでも文字化けしません。
