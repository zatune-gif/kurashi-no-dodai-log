### 【Section 1：Claude Codeとは】（20分）

**（1）4軸マップでClaude Codeの位置確認（3分）**

- 「ChatGPT・Geminiのような会話型AIと同じ分類に見えますが、Claude Codeはエージェント型です」
- 4軸マップを指しながら：「今日はこのエージェント領域の体験をします」

**（2）ツール別詳細表（3分）**

- 「Claude CodeはClaudeが作った専門ツールです。PCのファイル操作に特化しています」
- 表を使って特徴を簡単に説明

**（3）他のAIとの違い（4分）**

- 「ChatGPTやGeminiは答えを返してくれます。Claude Codeは答えるだけでなく、実際にファイルを作ったり動かしたりします」

**（4）ClaudeとClaude Codeの違い・利用条件・APIの説明（10分）**

| | Claude（claude.ai） | Claude Code |
|---|---|---|
| 種類 | 会話型AI | AIエージェント |
| 使い方 | ブラウザでチャット | デスクトップアプリ |
| できること | 答える・書く・提案する | ファイルを作る・動かす・整理する |
| イメージ | 優秀な相談相手 | 実際に手を動かす部下 |

- **できること：** 「ファイル・フォルダ操作 / CSV・データ整形 / 定型文書の一括生成」
- **できないこと：** 「ネット上での操作（注文・送信など）/ 完全な自動実行（毎回指示が必要）/ 複雑なシステム連携」
- **利用条件：** 「PC（Windows/Mac）+ Claude Codeデスクトップ版（無料） + Anthropicアカウント + Proプラン（$20/月・約3,000円）」
- **APIとは：** 「他のシステムからClaudeを呼び出す仕組みです。今日の研修では不要です。ExcelにClaudeのボタンをつけたい、自社アプリに組み込みたい、という場合に必要になります。その時は伴走サービスへどうぞ」
- **ハルシネーション対策：** 「Claude Codeも間違えます。CLAUDE.mdというメモファイルに『わからない場合は確認すること』と書いておくと大幅に減ります。後半で実践します」

### 【Section 4：応用ワーク】（40分）

**（1）CLAUDE.mdの作成（15分）**

- 「CLAUDE.mdはClaudeへの設定ファイルです。毎回同じことを言わなくて済むようになります」
- 「.mdはMarkdownという記法のファイルです。普通のテキストとして読めます」
- 一般的な記載内容を実演：

```
# 事業者情報
会社名：○○株式会社
業種：製造業
所在地：広島県

# 出力ルール
- 出力は必ず日本語で
- 箇条書きを優先する
- ファイルはデスクトップに保存すること

# 禁止事項
- 個人情報・機密情報は扱わない

# よく使う形式
- 文書はWord(.docx)
- データはCSV

# ハルシネーション対策
- わからない場合は作業を止めて確認すること
- 実行前に何をするか説明してから動くこと
```

- 「各自の会社情報・業務ルールに合わせて書き換えながら作成してみてください」

**（2）テンプレート化の説明→CLAUDE.mdに追記（10分）**

- 「テンプレートとは、よく使う指示文を記録しておく機能です。[ ] で変数にしておきます」
- 例：「お礼メールを作ってください。宛先：[社名]、用途：[用途]」
- 「CLAUDE.mdに『よく使う指示文』として追記しておくと、次回から呼び出すだけです」
- 「実際に今日使った指示文をCLAUDE.mdに追記してみてください」

**（3）AIが止まるケースと復旧方法（5分）**

持ち帰り資料の表を使って説明：

| トラブル | 対処 |
|---|---|
| 返答が途中で止まった | 「続けてください」と入力 |
| 「続けてよいですか？」と聞いてきた | 内容確認してOKなら「はい」と入力 |
| 意図と違う処理をした | 「待って」→ 意図を正確に伝え直す |
| エラーメッセージが出た | メッセージを貼り付けて「これはどういう意味ですか？」と聞く |
| 返答が遅い・繰り返す | 「ここまでの作業内容をまとめてください」→新規セッションで継続 |
| 応答しない・フリーズ | 数分待ってアプリ再起動 |
| どうしてもわからない | ブラウザ版Claude（claude.ai）に状況を説明して解決策を聞く |

- 「完璧に動かなくて大丈夫です。止まったら必ずこの7パターンのどれかで回復できます」
